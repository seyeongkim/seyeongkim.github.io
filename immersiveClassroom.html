Immersive Classroom

2015.September - 2016.Febuary

This is a collaborative research project with d'Strict and Jeju Marine Science Museum. We design 4 types of interactive learning solutions for children's marine museums.


01 Aqua Band

We design this interactive table for kids to make musics with tagible interface. Kids can play with underwater animal sound tags on the table and figure out how the music changes responding to the tags on real time

02 Imaginary Creature

This is an interactive table on which kids can make their own creatures while learning about characteristics of sea creatures

03 School of Fish

This is an interactive group game where kids solve quizzes together wearing interactive hats.

04 Layers of The Ocean

This is an interactive shooting game where children can learn about layers of the ocean and diffrent kinds of sea creatures in each layer.
